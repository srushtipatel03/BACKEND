Hyper Text Markup Language
What is HTML? HTML stands for Hyper Text Markup Language. 
HTML is the standard markup language for creating Web pages. 
HTML describes the structure of a Web page.

Why do HTML is used?
HTML is the language used to tell your web browser what each part of a website is. 
So, using HTML, you can define headers, paragraphs, links, images, 
and more, so your browser knows how to structure the web page you're looking at.
